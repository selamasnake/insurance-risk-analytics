# Notebooks Directory

This folder contains Jupyter notebooks for interactive exploratory data analysis on the insurance portfolio dataset.

## Key Notebooks

- `eda.ipynb`  
  The primary notebook demonstrating the full EDA workflow including:  
  - Data loading and preprocessing  
  - Descriptive statistics and data quality assessment  
  - Visualizations for univariate, bivariate, and segmentation analyses

## Usage

Open the notebooks in a Jupyter environment, ensure dependencies are installed, and run cells sequentially to reproduce the analysis.

---

Notebooks rely on the Python scripts in the `scripts/` directory for modular and reusable code components.
